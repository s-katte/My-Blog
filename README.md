## Steps to run the app

1. Clone the repository
2. go to the root directory
3. Install dependencies
```ruby
bundle install
```
4. Update database creadentials in `config/database.yml`
5. Create database
```ruby
rails db:create
```
6. Migrate database
```ruby
rails db:migrate
```
7. Spin up the server
```ruby
rails server
```
8. To run the test cases
```ruby
rspec
```
